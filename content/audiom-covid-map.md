# Audiom COVID-19 State Statistic Map

If you’re new to Audiom, please [complete this interactive Audiom tutorial first, then return to this page (Opens in a new tab)](https://www.audiom.net/embed/38?apiKey=b_e0xbMW4_hk9tDtOYMrZ&demo=true). Activate the “Start Audio Map” button or hit enter on the “Application” element (if using NVDA), and use the arrow keys to move. Press “t” to open a table of data, and activate any cell to jump to that state and switch the map to that statistic. The default step-size is around 70km each arrow press. There is a bug in Audiom that says the step size is 1 meter. [Please give us any feedback you have!
